003 : Web scraper with Python

Scraper for the hackernews.
Gets all the news and returns the titles and the links to the news in ascending order (sorts the articles on points). The script can be ran for one or more pages with news.

Source code: GitHub repo

The project has been built as part of the Complete Python Developer in 2019: Zero to Mastery Udemy course.
